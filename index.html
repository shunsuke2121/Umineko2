URL / anon key はどこにある？

Supabase（無料）でプロジェクトを作成 → 左メニュー Project Settings → API。
「Project URL」と「Project API keys」の anon public をコピー。
さらに初回のみ、SQL Editor にテーブル作成SQL（別途お渡し）を貼って実行してください。
